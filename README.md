MERN (JS) Setup

Server
- cd server
- copy .env.example to .env and adjust values
- pnpm i / npm i
- npm run dev

Client
- cd client
- npm i
- npm run dev

Auth Endpoints
- POST /api/auth/signup { name, email, password }
- POST /api/auth/login { email, password }

Client Routing
- /login, /signup, /dashboard
